## 🧩프로젝트 소개
'GiphyNameIs'는 인스타그램에서 사용된 **Giphy 이미지의 이름을 알려주는 크롬 확장 프로그램**입니다. </br>
사용자가 마우스 커서를 Giphy 이미지 위에 올리면, 해당 이미지의 이름을 화면에 표시해줍니다. 이를 통해 사용자는 자신이 사용하고 싶은 Giphy 이미지의 이름을 간편하게 얻어 사용할 수 있습니다.
